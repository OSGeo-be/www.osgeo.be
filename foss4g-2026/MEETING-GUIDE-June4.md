## Registration — free, Pretix-powered

- **Attendance is free** — required for logistics (capacity, catering, badges)
- **Training certificate** needed? Possible via organisational sponsorship — contact board@osgeo.be
- **Platform: Pretix** (open source, EU-hosted, embeds directly on the website)
  - Now: join the mailing list or Pretix waiting list for early notification
  - September: registrations open once programme is published → Pretix sends personal links automatically to everyone on the waiting list

## How to contribute on GitHub

You don't need to be a developer. Everything is plain text files (Markdown).

1. **Read**: no account needed — just open any link above
2. **Suggest**: create a GitHub account (free) → open an [Issue](https://github.com/OSGeo-be/www.osgeo.be/issues) to propose something
3. **Edit**: fork the repo, edit a file, open a Pull Request
